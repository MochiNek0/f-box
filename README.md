# F-Box: 极简 Flash 游戏浏览器

F-Box 是一款基于 Electron 开发的轻量级浏览器，专为现代系统中的 Flash 游戏体验而设计。它解决了现代浏览器停止支持 Flash 的痛点，提供简洁、高效且功能丰富的 Flash 游戏运行环境。

---

## 核心功能

### 基础浏览

- **🚀 原生 Flash 支持** — 自动扫描系统中的 Pepper Flash 插件，无需繁琐配置
- **📑 多标签页浏览** — 现代标签页设计，支持同时运行多个游戏，快速切换
- **🪟 智能弹窗接管** — 自动接管弹窗游戏（如赛尔号、洛克王国）
- **📁 托盘后台** — 最小化至系统托盘，随时待命

### 个性化体验

- **🕵️ 隐私老板键** — 全局快捷键（默认 `Esc`）一键隐藏/呼出窗口
- **🌫️ 窗口透明度** — 全局窗口透明度调节，与桌面背景完美融合
- **⚙️ 可视化设置面板** — 轻松调节各项参数

### 游戏辅助（Windows）

- **🎮 游戏库管理** — 内置精选游戏入口，支持自定义添加游戏链接
- **🕹️ 键位映射** — 基于 AutoHotkey，支持键盘重映射及手柄模拟

---

## 高级功能

### ⚡ 变速齿轮（Windows）

通过 DLL 注入 Hook Windows 计时 API，改变游戏时间流速。

**支持的挡位：**

| 挡位 | 用途 |
|------|------|
| 0.25x / 0.5x | 慢动作解析 |
| 1x | 正常速度 |
| 2x / 5x / 10x | 加速通关 |
| 100x | 快速跳过 |

- 支持自定义倍速输入（0.01 ~ 999）
- 可视化速度条，实时显示当前状态
- **提示**：启动前请先加载 Flash 游戏；切换游戏页面需重新启动

### 🤖 自动化工具（Windows）

- **📹 操作录制/回放** — 录制鼠标点击、键盘输入，精确到毫秒级时间戳，支持断点暂停
- **👁️ OCR 图像识别** — 内置 Paddle OCR 引擎，支持屏幕文字识别和条件判断
- **🖱️ 连点器** — 自定义点击频率、按键类型（左键/右键/双击）

## 快速开始

### 环境要求

- Node.js 18+
- Windows/macOS
- Flash 插件（自动检测）

### 安装运行

```bash
npm install
npm run dev
```

### 打包构建

```bash
# Windows 安装程序
npm run dist:win

# Windows 便携版
npm run dist:win:portable

# macOS
npm run dist:mac
```

## 使用指南

### 变速齿轮

1. 打开设置面板 → 「变速」标签页
2. 先加载 Flash 游戏页面
3. 点击「启动变速」
4. 选择预设挡位或输入自定义倍速
5. 切换游戏后需重新启动变速

### 自动化脚本

1. 在「操作自动化」标签页点击「新建录制」
2. 录制器最小化为工具条，点击开始捕捉操作
3. 输入脚本名称保存
4. 在自动化列表中点击播放回放

### OCR 功能

- 首次使用自动下载 Paddle OCR 模型（约 200MB）
- 脚本断点处可手动框选识别区域
- 识别结果用于条件判断

## 常见问题

**Q: 找不到 Flash 插件？**
A: 确保系统安装了 PPAPI 版 Flash Player。F-Box 自动寻找 `pepflashplayer*.dll`。

**Q: 变速齿轮无法启动？**
A: 请将应用目录加入杀毒软件白名单，DLL 注入可能被拦截。

**Q: OCR 功能异常？**
A: 首次使用需下载 OCR 模型，确保网络连接正常。
